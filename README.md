### INSTALL
```bash
$ pip install hello-pysatellite
```

### DEV
```bash
$ git clone ...
$ cd hello-pysatellite
$ pdm venv create
$ source .venv/bin/activate
(hello-pysatellite-3.8) $ pdm install
```

### DEPLOY 
```bash
$ pdm publish
```
