### DEV
```bash
$ git clone ...
$ cd hello-pysatellite
$ pdm venv create
$ source .venv/bin/activate
(hello-pysatellite-3.8) $ pdm install
```

### TEST
```bash
$ pdm add -dG test pytest pytest-cov
$ pytest
$ pytest -s
$ pytest --cov

```

### DEPLOY 
```bash
$ pdm publish
```
